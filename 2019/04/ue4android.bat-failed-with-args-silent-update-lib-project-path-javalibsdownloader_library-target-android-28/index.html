[UE4]android.bat failed with args --silent update lib-project --path JavaLibsdownloader_library --target android-28

April 27, 2019 in UnrealEngine4

keywords: android sdk, UE4 package error

Error when Android Packaging:

LogPlayLevel: Program.Main: ERROR: AutomationTool terminated with exception: ERROR: C:/NVPACK/android-sdk-windows/tools/android.bat failed with args --silent update lib-project --path JavaLibs/downloader_library --target android-28

Solution:
Download this archieve
https://drive.google.com/file/d/1J9qaXChcA15govsyK35ZwckMEUyIZ8xx/view?usp=sharing
then extract tools.rar to replace same directory in android-windows-sdk
