Thursday, 22:13, Mar 28, 2019 in UnrealEngine4
How to increase slider bar thickness (height, scale in Y)

3 Ways

  1. Modify Scale Y of Render Transform
    Increase Scale Y

    Then decrease Size Y of Thumb Then decrease Size Y of Slot Result

  2. Use ScaleBox
    Put Slider into ScaleBox

    Set Stretch to Scale to Fit Y Change Horizontal Alignment and Vertical Alignment Result Or you can set Stretch to User Specified and Increase User Specified Scale

  3. Hide background image (Thumb)

    Then add an customized image down of Slider
