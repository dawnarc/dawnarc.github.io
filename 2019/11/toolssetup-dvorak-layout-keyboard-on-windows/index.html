[Tools]Setup Dvorak layout keyboard on Windows

Saturday, 15:10, Nov 2, 2019 in Tools

Keywords: Dvorak layout keyboard

1, Input regedit in Start / Run(Win + R)

2, Address to following primary key:

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Keyboard Layouts\E0040804]

3, You’ll see following informations:

"Layout File"="kbdus.dll"
"Layout Text"="中文 (简体) - 智能 ABC"
"IME File"="winabc.ime"

Change kbdus.dll to KBDDV.DLL, then save and restart system.

4, New a binary option named ScanCode Map under \HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control\Keyboard Layout.

5, Copy following content to a text file and name it as dvorak.reg, double click it and restart system.

Windows Registry Editor Version 5.00
[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Keyboard Layout]
"ScanCode Map"=hex:00,00,00,00,00,00,00,00,22,00,00,00,2d,00,30,00,24,00,2e,00,\
11,00,33,00,33,00,11,00,12,00,20,00,34,00,12,00,1b,00,0d,00,0d,00,1b,00,16,\
00,21,00,17,00,22,00,20,00,23,00,1a,00,0c,00,2e,00,17,00,23,00,24,00,14,00,\
25,00,31,00,26,00,35,00,1a,00,30,00,31,00,13,00,18,00,26,00,19,00,2f,00,34,\
00,28,00,10,00,0c,00,28,00,19,00,13,00,18,00,1f,00,1f,00,27,00,2c,00,35,00,\
15,00,14,00,22,00,16,00,25,00,2f,00,10,00,2d,00,21,00,15,00,27,00,2c,00,00,\
00,00,00

Or double click qwerty.reg(following contents) if want to recover to qwerty layout keyboard.

Windows Registry Editor Version 5.00
[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Keyboard Layout]
"ScanCode Map"=hex:00,00,00,00,00,00,00,00,01,00,00,00,00,00,00,00
