[MySQL]Windows上安装MySQL5.7绿色版

Saturday, 13:59, May 27, 2017 in DB

keywords: MySQL, Windows, 绿色版, 安装

下载：
https://dev.mysql.com/downloads/mysql/

解压后执行（管理员命令行）：

mysqld --initialize-insecure --user=mysql

复制一份里面的my-default.ini为my.ini，（如果zip没有my-default.ini，则手动创建一个my.ini）

[mysqld]
basedir = "D:/mysql-5.7.18-winx64"
datadir = "D:/mysql-5.7.18-winx64/data"
port = 3306
character-set-server = utf8
max_connections = 200
default-storage-engine = INNODB
 
[mysql]
default-character-set = utf8

然后安装windows服务：

mysqld install MySQL --defaults-file="C:\ProgramData\MySQL\MySQL Server 5.7\my.ini"

如果不需要ini配置文件（不推荐），可以不设置defaults-file：

mysqld install

启动：

net start mysql

修改root密码（首次安装后root还未修改的情况下，即密码为空）

mysqladmin -u root -p password 123456

参考文章：
win10下zip安装mysql5.7的一些问题
https://www.cnblogs.com/jkko123/p/6294573.html

my.ini找不到

Windows 10下，安装MySQL后的my.ini文件位置不再是安装目录的根目录，而是：C:\ProgramData\MySQL\MySQL Server 5.7

Windows Services移除MySQL

CMD:

SC DELETE mysql
