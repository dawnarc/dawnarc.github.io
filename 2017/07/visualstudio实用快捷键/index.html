[VisualStudio]实用快捷键

Friday, 14:51, Jul 21, 2017 in VisualStudio
Visual Studio 默认快捷键

vs光标上下移动的快捷键，光标定位到函数第一行。

Ctrl + ]，在一对大括号的首位之间跳转。

Alt + ]，选中当前括号内的内容，按住Alt不放，连续按]多次，可以一层一层向外选择括号的所有内容。

Alt +上下键，光标在同级别的大括号中上下跳转。

Ctrl + M + O 折叠所有函数体

Ctrl + M + L 展开所有函数

Ctrl + Shift + 上下键 当前打开的代码中，光标所在的变量名挨个向上或向下跳转

Ctrl + Back Space，当前行光标到上一个空格之间的字符串全部删除。

Ctrl + Delete，当前行光标到下一个空格之间的字符串全部删除。

Alt + Shift + T, 当前行和下一行交换

F3, 想下查找Ctrl+F中的字符串

F12 从头文件中变量或函数申明处，CPP中变量或函数定义处，

Ctrl + K + O : 在头文件和CPP文件之间跳转。

VS 2019 快捷键：

Alt + F12 当光标在CPP文件中时，打开光标当前所处位置的变量或函数的对应头文件（浮动窗口中打开，此时可以同时查看CPP和头文件）

Visual Assist 快捷键

Alt + O，在头文件和cpp之前跳转

Alt + G，在函数的头文件和cpp文件之间跳转

VAssistX.OpenFileInSolutionDialog Alt + Shift + O

VAssistX.FindSymbolDialog Alt + Shift + S

VAssistX.ResetEditorZoom Ctrl + 0

VAssistX.ScopeNext Alt + Down Arrow

VAssistX.ScopePrevious Alt + Up Arrow

VAssistX.NavigateBack Alt + Left Arrow

VAssistX.NavigateForward Alt + Right Arrow

VAssistX.GotoRelated Alt + Shift + G

VAssistX.GotoImplementation Alt + G

VAssistX.SmartSelectShrink Alt + Shift + [

VAssistX.SmartSelectShrinkBlock Alt + [
