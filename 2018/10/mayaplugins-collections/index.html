Maya Animation Plugins

Maya Plugins & Scripts for Animation
https://www.youtube.com/watch?v=x8gEfLzBcHI

  1. Animbot Animbot is a set of tools for Maya aimed at speeding up workflow. it has a lot of tools that can make the animator’s daily tasks feel like creative work rather than doing a chore.

  2. Ziva VFX Ziva Dynamics the developer of Ziva products was founded in early 2015 by Academy Award-winner James Jacobs and Dr. Barbic .

  3. MGTools MG Tools can save time and create a more efficient workflow. Miguel Winfield’s plugin MG Tools has a number of features that aim to make things simpler and easier for the Maya animators.

  4. Advanced Skeleton Advanced Skeleton can create rigs with unlimited body configurations,. Not only creatures, but you can also rig props, vehicles, and just about anything. You can at any time go from Advanced Skeleton, back to the Fit Skeleton, make changes, and rebuild.

  5. bh ghost This is an animation tool Brian Horgan wrote for Maya that provides a different approach to ghosting/onion-skinning in 3D animation.

  6. TweenMachine Tween Machine was originally developed by Justin Barrett and used extensively throughout the animation and visual effects industries for years.

  7. Physics tools Maya physics tools allow you to spend more time on creating appealing poses that better show human emotions and movements rather than technical issues that Animators usually spend most of their time on.

  8. Overlapper overlapper is a tool created by Dmitrii Kolpakov which can make overlapping action type of animation easier by eliminating all the work that animators need to make overlapping actions believable.

  9. Craft Director Studio With this animation plug-in, you simply hook up to a model and start “driving” vehicles and aircraft through3D environments. Adjust variables like suspension and add camera effects to achieve the exact look and feel you’re after.

  10. bh aim tools bh_aimTools is an animation tool Brian Horgan created for Maya, inspired by the workflow demos of Richard Lico. It streamlines the process of using aim targets to control rotation animation and applying the animation back to the controls.

  11. bh path Anim bh_pathAnim is a flexible tool for path animation in Maya. The goal behind path Anim was to make it simple and intuitive to attach any rig, with any number of limbs and body controls, to a path and animate it walking or running along with it.

  12. Studio Library it is a free tool for managing poses and animations in Maya, Studio Library is able to store animations so if you’ve got poses, you can select sets and click on specific poses, and it brings it straight up.

  13. bh waveit bh_waveIt is an animation tool for Maya which speeds up the process of blocking in simple wave motion for things like tails, capes and anything else that might need it.
