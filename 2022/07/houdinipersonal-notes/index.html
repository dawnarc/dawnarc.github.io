Solution:
Delete old rbdconfigure node and add a new rbd configure node.

Unreal Engine

Attribute Create doesn’t work in Unreal Editor

Case:
unreal_material doesn’t work when build static mesh assets.

Solution:
Place Transform node after Attribute Create node.

it’s an issue of HoudiniEngine 19.0 version.
