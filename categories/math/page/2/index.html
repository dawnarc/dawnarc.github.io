[Math]如何检测某个点是否在多边形内部或者直线上

点在多边形内的判别方法 int pnpoly(int nvert, float *vertx, float *verty, float testx, float testy) { int i, j, c = 0; for (i = 0, j = nvert-1; i < nvert; j = i++) { if (((verty[i] > testy) != (verty[j] > testy)) && (testx < (vertx[j] - vertx[i]) * (testy - verty[i]) / (verty[j] - verty[i]) + vertx[i]) ) { c = !c; } } return c; } Arguments nvert: Number of vertices in the polygon.
