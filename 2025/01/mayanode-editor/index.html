[Maya]Node Editor

Monday, 19:53, Jan 6, 2025 Maya

Keywords:Maya, Editor, Node Editor

Hotkey

F: Center the node to the screen.

Common Tricks

Select target node in viewport or hierarchy tree view, then click Add selected nodes to graph, and you can click Remove selected nodes from graph to remove unused node.

Click Clear the graph to clean the screen, it doesn’t remove any node ,just hide them.

You can pin a node, then it will not be cleared when clearing screen.

Click Layout the graph if nodes was messed.

You can filter shape node: Display -> No Shapes.

If there are too many wires in the graph, you can clean what you not needed.
And you can re-add them again by click Output Connections.

Hotkey to expand or collapse node: Num 3, Num 2, Num 1.

If the Node Editor is open when the Node is created, there will be a line connecting it from the beginning to the end.

Q: How to clean this line?
A: Remove selected node from graph, then drag node from Outliner to Node Graph.

Issues

If import controllers from other projects, the old controller may contains dirty transformation that didn’t shown in Channel Box and Attribute Editor.

Solution:
Rename controller, then connect to joint (e.g. build connnection from World Matrix to Offset Parent Matrix), then rename controller to the original name again.
